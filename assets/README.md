# Brand Assets

Place the following files in this directory:

## Required Files

### 1. codenest-logo.png
- **Purpose:** PDF report header and cover page
- **Recommended Size:** 200x80px or similar aspect ratio
- **Format:** PNG with transparent background preferred

## Configuration

After uploading the logo, the app will automatically use it in:
- PDF cover page (centered, large)
- PDF header (left aligned, smaller)

For email logos, update `EMAIL_LOGO_URL` in `app.py` with your hosted logo URL.
